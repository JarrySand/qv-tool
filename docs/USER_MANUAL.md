## Quadratic Voting とは

Quadratic Voting（二次投票）は、意思決定における「選好の強さ」を反映できる投票方式です。

### 基本ルール

- 各投票者には一定の **クレジット** が与えられます
- 投票対象に票を投じると、**票数の2乗** のクレジットを消費します

| 投じる票数 | 消費クレジット |
|------------|----------------|
| 1票 | 1クレジット |
| 2票 | 4クレジット |
| 3票 | 9クレジット |
| 4票 | 16クレジット |
| 5票 | 25クレジット |

### なぜ二次投票？

従来の「1人1票」方式では、強い関心を持つ少数派の意見が反映されにくい問題がありました。

二次投票では：
- 本当に重要な選択肢には多くの票を投じられる（ただしコストが高い）
- 複数の選択肢に分散して投票することも可能
- 少数派の強い意見も適切に反映される

## イベント主催者向け

### イベントの作成

1. **トップページにアクセス**
   - [QV-Tool](https://your-app.vercel.app) にアクセス

2. **「イベントを作成」をクリック**

3. **イベント情報を入力**
   
   | 項目 | 説明 |
   |------|------|
   | タイトル | イベントの名前（必須） |
   | 説明 | イベントの詳細説明（任意） |
   | カスタムURL | 短くて覚えやすいURL（任意） |
   | 開始日時 | 投票開始日時 |
   | 終了日時 | 投票終了日時 |
   | クレジット数 | 1人あたりのクレジット（デフォルト: 100） |
   | 認証方式 | 投票者の認証方法を選択 |

4. **認証方式の選択**

   | 方式 | 説明 | 向いている用途 |
   |------|------|----------------|
   | 個別URL方式 | 参加者ごとにユニークなURLを発行 | 限定された参加者、匿名投票 |
   | Google | Googleアカウントでログイン | 一般公開、重複防止が重要 |
   | LINE | LINEアカウントでログイン | 日本向け、LINEグループ |

5. **「イベントを作成」をクリック**

6. **⚠️ 重要: 管理用URLを保存**
   - 作成完了画面に表示される **管理用URL** を必ず保存してください
   - このURLは二度と表示されません
   - このURLがあればイベントを編集・管理できます

### 投票対象の追加

1. **管理画面にアクセス**
   - 保存した管理用URLにアクセス

2. **「選択肢を追加」をクリック**

3. **情報を入力**
   - タイトル（必須）
   - 説明（任意）
   - 画像URL（任意）
   - リンクURL（任意）

4. **「保存」をクリック**

5. **必要な数の選択肢を追加**

> ⚠️ **注意**: 投票が開始されると（誰かが投票すると）、選択肢の追加・編集・削除はできなくなります。

### 参加者の招待

#### 個別URL方式の場合

1. 管理画面で「トークンを生成」をクリック
2. 必要な数のトークンを生成
3. 「CSVをダウンロード」で参加者リストを取得
4. 各参加者に専用のURLを配布

#### Google/LINE認証の場合

1. 管理画面から「公開URL」をコピー
2. 参加者にURLを共有
3. 参加者はログイン後に投票可能

### 結果の確認

1. **結果ページにアクセス**
   - 公開URL + `/result` または管理画面から「結果を見る」

2. **表示される情報**
   - 各選択肢の得票数（グラフ）
   - 順位
   - 統計情報（参加者数、消費クレジットなど）
   - 投票分布

3. **データのエクスポート**（管理者のみ）
   - 集計データ: 順位・得票数・消費クレジット
   - ローデータ: 個別の投票データ

## 投票者向け

### 投票に参加する

#### 個別URL方式の場合

1. 主催者から受け取ったURLにアクセス
2. ログイン不要でそのまま投票ページへ

#### Google/LINE認証の場合

1. イベントのURLにアクセス
2. 「投票する」をクリック
3. Google または LINE でログイン
4. 投票ページへ

### 投票の仕方

1. **投票ページの構成**
   - 上部: 残りクレジットと消費クレジット表示
   - 中央: 投票対象の一覧
   - 下部: 投票ボタン

2. **票の投じ方**
   - 各選択肢の「+」ボタンで票を増やす
   - 「-」ボタンで票を減らす
   - 票を増やすとコストが表示される

3. **クレジットの確認**
   - 残りクレジットがリアルタイムで更新
   - プログレスバーで使用状況を確認
   - クレジットが足りないと「+」ボタンが無効に

4. **投票の送信**
   - 内容を確認して「投票する」をクリック
   - 完了画面が表示される

### 投票の変更

投票期間内であれば、何度でも投票を変更できます。

1. イベントページにアクセス
2. 「投票を変更」をクリック
3. 票数を調整
4. 「投票を変更」をクリック

## よくある質問

### Q: クレジットを使い切る必要がありますか？

**A**: いいえ、必須ではありません。使わなかったクレジットは消滅します。重要だと思う選択肢に集中して投票することをお勧めします。

### Q: 同じ選択肢に複数票を投じるべきですか？

**A**: 強く支持する選択肢があれば、複数票を投じることで意思を強く反映できます。ただし、コストが急激に増加するため、バランスを考えましょう。

### Q: 投票結果はいつ見られますか？

**A**: リアルタイムで公開されます。投票期間中でも結果を確認できます。

### Q: 管理用URLを紛失しました

**A**: 残念ながら、管理用URLは再発行できません。セキュリティ上の理由から、作成時に一度だけ表示されます。

### Q: 投票期間を延長したい

**A**: 管理画面から終了日時を変更できます。ただし、投票が開始されている場合、一部の設定は変更できません。

### Q: マイナス票（反対票）は投じられますか？

**A**: いいえ、QV-Tool では0以上の票のみ投じられます。これは設計上の決定です。
